Generates a sin wave signal that could be fed into the component chain.

Parameters
sampling_frequencyThe frequency at which the wave is sampled. Consecutive samples are separated by units of time equalling 1/sample_frequency
sine_wave_frequencyThe frequency with which the wave oscillates. Two wave samples separated by units of time equalling 1/sine_wave_frequency will have the same value
measurement_periodThe number of time units to generate the wave for
phase_timeNumber of time units the phase of the wave will be delayed by
Returns
A numpy array length measurement_period*sampling_frequency containing the sine wave samples
